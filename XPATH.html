XPATH

Introduccion

XPATH es un lenguaje que permite construir expresiones que recorren y procesan un documento XML. La idea es parecida a las expresiones regulares para seleccionar partes de un texto sin atributos. XPATH permite buscar y seleccionar teniendo en cuenta la estructura jerarquica del XML (Es necesario conocer este lenguaje para el uso de XSLT y XQUERY)
